Quais os comandos para configurar usuário e email no git?

git config --global user.name
git config --global user.email Todos os dados deverão estar entre aspas "usuario" - "usuario!email.commit"

Qual a diferença entre git e github?

Git é uma tecnologia de versionamento. O gitHub é um sistema que hospeda arquivos e códigos com controle de versionamento.

O que faz o comando commit

Prepara os arquivos e suas alterações que você deseja colocar no repositório.
Estes arquivos, no qual passaram pelo commit são semelhantes a um recorte do estado atual dos seus arquivos.

O que faz o comando push

Sobe os arquivos preparados com o commit para o servidor

O que faz o comando pull

Ele verifica se o repositório selecionado possui alguma alteração (commit) no servidor, se possuir ele realiza o download da alteraçãp e atualiza no seu repositório local.
